# JavaScript Basics

This repository contains beginner-friendly JavaScript notes and practice programs. It covers the fundamental concepts of JavaScript with simple examples to help beginners learn step by step.

 📚 Topics Covered

- Introduction to JavaScript
- Variables
- Operators
- If-Else Conditions
- Loops
- Functions
- Strings
- Arrays
- Objects
- Array Methods (Map, Filter, etc.)

 🎯 Purpose

The purpose of this repository is to help beginners understand JavaScript basics through easy-to-follow examples and practice programs.

 🛠 Technologies Used

- JavaScript (ES6+)

 💻 How to Use

1. Clone this repository.
2. Open the project in VS Code.
3. Run the JavaScript files using Node.js or your browser console.
4. Practice and modify the examples to improve your understanding.
